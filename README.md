# 🌤️ App de Clima

Projeto simples em JavaScript para buscar dados de clima usando a [API Open-Meteo](https://open-meteo.com/).

## 🚀 Funcionalidades
- Buscar temperatura por cidade;
- Consumo de API externa.

## 🛠️ Tecnologias
- JavaScript;
- HTML + CSS;
- Open-Meteo API.

Projeto em desenvolvimento (aprendizado).
